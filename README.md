## Requirements

- PHP ^8.0
- NodeJS ^18.0
- SQL Database

## Setup

```bash
git clone https://github.com/mestrey/alexandriya.git
cd alexandriya
```

### Backend

```bash
cd backend
cp .env.example .env
composer install

# start a local server with a db
```

Edit the .env file with your data

### Frontend

```bash
cd frontend
npm install
npm run dev
```
